### Direct proof of Γ = Γ(a) · Γ(N) when gcd(a,N) = 1

For γ ∈ SL₂(ℤ), we need γ_a ∈ Γ(a), γ_N ∈ Γ(N) with γ = γ_a · γ_N.

**Step 1**: By CRT (entry-wise), find β ∈ M₂(ℤ) with β ≡ I mod a, β ≡ γ mod N.
Then det(β) ≡ 1 mod a and det(β) ≡ 1 mod N, so det(β) ≡ 1 mod aN.

**Step 2**: Need to adjust β to have det = 1 exactly (not just mod aN).
Write det(β) = 1 + k·aN.

**Key lemma** (surjectivity of SL₂(ℤ) → SL₂(ℤ/dℤ)):
For any A ∈ M₂(ℤ) with det(A) ≡ 1 mod d, there exists τ ∈ SL₂(ℤ)
with τ ≡ A mod d.

Proof of key lemma:
(a) SL₂(𝔽_p) is generated by images of T and S⁻¹TS (for prime p)
(b) SL₂(ℤ/p^kℤ) surjects from SL₂(ℤ) by Hensel lifting
(c) CRT gives surjectivity for composite d

With the key lemma: apply it to β and d = aN. Get τ ∈ SL₂(ℤ) with
τ ≡ β ≡ I mod a and τ ≡ β ≡ γ mod N.
Set γ_a = τ (then τ ∈ Γ(a)) and γ_N = τ⁻¹γ (then τ⁻¹γ ≡ I mod N, so γ_N ∈ Γ(N)).
γ = γ_a · γ_N. ✓
