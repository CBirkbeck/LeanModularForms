## Source: Shimura, *Introduction to the Arithmetic Theory of Automorphic Functions*,
Chapter 3, §3.2–3.3 (pp. 55–71)

### Crucial structural observation

Shimura's proof **does not** compute `deg_Γ₀(T'(1, p^k))` directly via CRT. Instead,
the whole architecture is based on **coset-level bijection** and then **polynomial
presentation**:

1. **Lemma 3.28** (`Γ(gcd(a,b)) = Γ(a)·Γ(b)`) — already in the code as
   `Gamma_gcd_eq_mul`.
2. **Lemma 3.29** (5 parts about coprime-det cosets) — partially in the code
   (parts of (3) are in the existing `doubleCoset_eq_of_Gamma0_coprimeDet`).
3. **Prop 3.30**: the map `Γ'αΓ' ↦ ΓαΓ` is a **ring** homomorphism
   `R(Γ', Φ) → R(Γ, Δ)`. Proof: use Lemma 3.29 (5) so the left-coset
   representatives are shared, then Lemma 3.29 (1) gives multiplicity-count
   equality for each target double-coset.
4. **Prop 3.31**: restricted to `R(Γ', Δ'_N) → R(Γ, Δ_N)` (coprime det on both
   sides), Prop 3.30 is an **isomorphism**. Injectivity is proved at
   `shimura_prop_3_31` and surjectivity at `shimura_prop_3_31_surjective` in
   CongruenceHecke.lean — the ring-level equivalence is the part still needed.
5. **Prop 3.32/3.33**: bad-prime structure. `T'(m) = deg m` for `m | N^∞`.
6. **Thm 3.34**: `R(Γ', Δ')` is a polynomial ring on
   `T'(p)` (p | N), `T'(1, p)`, `T'(p, p)` (p ∤ N).
7. **Thm 3.35**: Define `T(n) ↦ T'(n)`, `T(p, p) ↦ T'(p, p)` (p ∤ N),
   `T(p, p) ↦ 0` (p | N). Well-defined via:
   - Thm 3.20 (GL polynomial presentation)
   - Thm 3.34 (Γ₀ polynomial presentation)
   - Prop 3.31 (iso on coprime-det generators)
   - The bad-prime relations are respected automatically (killing T(p,p)
     doesn't break any polynomial relation).

### What this means for our Lean formalization

**No explicit degree computation of `deg_Γ₀(T'(1, p^k))` is needed.** The degrees
match automatically via Lemma 3.29 (5): if we pick the same SL₂-coset reps
`{α_i}` for `Γ'αΓ'` (which Lemma 3.29 (5) guarantees work), then the left-coset
count is identical.

**No CRT for Γ₀(N)/Γ₀(mN)** is needed. No `[SL₂ : Γ₀(p^k)]` computation is
needed. No second isomorphism theorem is needed.

The whole architecture collapses into Lemma 3.29 + Prop 3.30 + Prop 3.31, plus
the bad-prime structure (Prop 3.32/3.33, already partially formalized via
`T_bad_mul`).

## Revised phases

### Phase A — Lemma 3.29: coset bijection structure on coprime det

**Objective**: formalize parts (1), (3), (4), (5) of Shimura Lemma 3.29.

#### A.1 — Lemma 3.29 (3): `ΓαΓ = Γ·α·Γ_N = Γ_N·α·Γ` for α ∈ Δ_N

**Location**: CongruenceHecke.lean, near `doubleCoset_eq_of_Gamma0_coprimeDet`.

**Statement** (as a set equality, which is what Shimura uses):
```lean
/-- Shimura Lemma 3.29 (3): For α with coprime det (gcd(det α, N) = 1),
    the SL₂-double coset equals the Γ_N-left coset map. -/
private lemma Lemma_3_29_3_left (N : ℕ) [NeZero N] (α : GL (Fin 2) ℚ)
    (A : Matrix (Fin 2) (Fin 2) ℤ)
    (hA : (↑α : Matrix _ _ ℚ) = A.map (Int.cast : ℤ → ℚ))
    (hA_det_pos : 0 < A.det)
    (hA_det_coprime : Int.gcd A.det N = 1) :
    DoubleCoset.doubleCoset α (SLnZ_subgroup 2 : Set _) (SLnZ_subgroup 2 : Set _) =
    DoubleCoset.doubleCoset α (SLnZ_subgroup 2 : Set _)
      (((CongruenceSubgroup.Gamma N).map (mapGL ℚ)) : Set _)
```

Similar lemma `Lemma_3_29_3_right` for the other side.

**Proof strategy** (following Shimura p. 66):
1. By Lemma 3.28 applied to `a = det α, b = N`, get `Γ(1) = Γ(a)·Γ(b)`, i.e.,
   `SL₂(ℤ) = Γ(a)·Γ(N)` (since `(a, N) = 1` and `Γ(1) = SL₂`).
2. Conjugate: `α · Γ(a) · α⁻¹ ⊂ SL₂(ℤ)` by `conj_ker_mem_SLnZ` (already in the code).
3. So `αΓ = αΓ(a)Γ(N) ⊂ SL₂·αΓ(N) ⊂ ΓαΓ(N)`.
4. Since the reverse inclusion is trivial, we get equality.

**Depends on**: `Gamma_gcd_eq_mul`, `conj_ker_mem_SLnZ`. Both already in the code.

**LOC estimate**: 40-80.

#### A.2 — Lemma 3.29 (5): Γ₀(N) and SL₂ share left-coset reps

**Statement**:
```lean
/-- Shimura Lemma 3.29 (5): if α has coprime det and
    Γ₀(N)αΓ₀(N) = ⋃_i Γ₀(N)·α_i is a disjoint union, then
    SL₂·α·SL₂ = ⋃_i SL₂·α_i is ALSO a disjoint union. -/
private lemma Lemma_3_29_5 (N : ℕ) [NeZero N]
    (α : (Gamma0_pair N).Δ) (hα : CoprimeDet N α)
    (reps : Finset (GL (Fin 2) ℚ))
    (hrep_mem : ∀ r ∈ reps, r ∈ DoubleCoset.doubleCoset (↑α : GL _ ℚ)
      ((Gamma0_pair N).H : Set _) ((Gamma0_pair N).H : Set _))
    (hrep_cover : ⋃ r ∈ reps, (r • (Gamma0_pair N).H.carrier : Set _) =
      DoubleCoset.doubleCoset (↑α : GL _ ℚ)
        ((Gamma0_pair N).H : Set _) ((Gamma0_pair N).H : Set _))
    (hrep_disj : reps.toSet.PairwiseDisjoint (fun r => r • (Gamma0_pair N).H.carrier)) :
    ⋃ r ∈ reps, (r • (SLnZ_subgroup 2 : Set _) : Set _) =
      DoubleCoset.doubleCoset (↑α : GL _ ℚ)
        (SLnZ_subgroup 2 : Set _) (SLnZ_subgroup 2 : Set _) ∧
    reps.toSet.PairwiseDisjoint (fun r => r • (SLnZ_subgroup 2 : Set _))
```

**Proof strategy** (Shimura p. 66):
1. Coverage: the SL₂ double coset equals the Γ₀(N) double coset by Lemma 3.29 (4),
   so the reps already cover the SL₂-DC. The SL₂-cosets `r · Γ` are unions of
   Γ₀(N)-cosets, so the cover follows.
2. Disjointness: suppose `Γ·α_i = Γ·α_j`, i.e., `α_i = γα_j` with `γ ∈ Γ = SL₂`.
   By (1), this implies `α_i ≡ δα_j (mod N)` with `δ ∈ Γ'`, hence `γ ≡ δ (mod N)`,
   hence `γ ∈ Γ'` (since `Γ_N ⊂ Γ'`), so `Γ' · α_i = Γ' · α_j`, so `i = j`.

**Depends on**: A.1 (for the SL₂-DC = Γ₀-DC step).

**LOC estimate**: 60-100.

### Phase B — Prop 3.30: Ring homomorphism structure

**Objective**: upgrade the existing `cosetMap` (additive) to a **ring hom** on
coprime-det elements.

#### B.1 — Same left-coset reps give same multiplicities

**Statement**:
```lean
/-- For coprime-det α, β, and ξ ∈ Δ_N: the number of pairs (i, j) with
    Γ₀αᵢβⱼ = Γ₀ξ equals the number with Γαᵢβⱼ = Γξ. -/
private lemma heckeMultiplicity_eq_on_coprime_det (N : ℕ) [NeZero N]
    (α β : (Gamma0_pair N).Δ) (hα : CoprimeDet N α) (hβ : CoprimeDet N β)
    (ξ : (Gamma0_pair N).Δ) (hξ : CoprimeDet N ξ) :
    HeckeRing.heckeMultiplicity (Gamma0_pair N) α β ξ =
    HeckeRing.heckeMultiplicity (GL_pair 2) (Delta0_inclusion N α)
      (Delta0_inclusion N β) (Delta0_inclusion N ξ)
```

**Proof** (Shimura p. 67):
1. By Phase A.2, pick shared reps `{α_i}, {β_j}` for both `Γ₀αΓ₀` and `ΓαΓ`
   (resp. β).
2. Use Lemma 3.29 (1): `Γ₀α_iβ_j = Γ₀ξ ⟺ Γα_iβ_j = Γξ ∧ α_iβ_j ≡ ξ (mod N)`.
3. Since `ξ ∈ Δ_N` has coprime det, and α, β ∈ Δ_N have coprime det, `α_iβ_j` also
   has coprime det, so the λ_N condition is automatic.
4. Hence the counts coincide.

**LOC estimate**: 80-120.

### Phase F — Prove T_gen_algebraicIndependent (π_injective)

**Objective**: the other dependency of `shimura_thm_3_35`: need `π_hom` to be
injective for `ker_π_le_ker_ψ` to become trivial.

`T_gen_algebraicIndependent` says that the GL Hecke algebra generators
`T_gen 2 p i` (for primes p and i ∈ Fin 2) are algebraically independent over
ℤ. This is Shimura Thm 3.20 generalized to all primes (the "tensor of p-locals").

**Proof strategy** (Shimura pp. 58-60):
1. `R_p^(n)` is polynomial ring on `T(p^{e_1}, ..., p^{e_n})` (Thm 3.20).
2. For n=2: `R_p^(2) = ℤ[T(1,p), T(p,p)]`.
3. `R(Γ, Δ) = ⊗_p R_p^(2)` (as a ℤ-algebra).
4. Global algebraic independence = local algebraic independence at each p.

**Stubbed helpers to unstub**:
- `coprime_mul_coeff`: factoring Finsupp coefficients across coprime supports.
- `T_gen_pow_support_qpower`: support structure of `T_gen(p,·)^e`.
- `T_gen_pow_entries_qpower`: entries are q-powers.
- `support_mul_exists`: support of product is nonempty if product is nonzero.
- `support_det_mul`: determinant multiplicativity.
- `prod_gen_det_eq`: determinant of generator product.
- `prod_gen_support_det`: support determinant.
- `multi_prime_coeff_factor`: prime-wise factoring of multi-prime coefficient.

These are all technical support-tracking lemmas for the Finsupp structure of
the Hecke algebra.

**LOC estimate**: 300-500.
